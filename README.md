# ⌨️ 磁轴键盘性能测试工具

一个专为《无畏契约》（Valorant）玩家设计的磁轴键盘性能测试工具，用于精确测量 WASD 按键的急停响应时间。

# 线上使用
访问网站可直接使用：http://www.supersaiyan.online:8088/

## 🎯 简介

在 FPS 游戏如《无畏契约》中，**急停**是一项核心技术。当你松开 WASD 移动键时，游戏角色需要立即停止移动才能精准射击。

本工具通过浏览器原生事件监听，精确测量从**按下按键到完全抬起**的系统响应时间，帮助你评估磁轴键盘的急停性能。

## ✨ 功能特性

### 🔥 急停专项测试
- **WASD 四键独立检测**：支持 W、A、S、D 四个方向键的独立急停测试
- **实时视觉反馈**：圆形指示器动态显示"移动"/"抬起"状态
- **智能评级系统**：
  - ⚡ **< 20ms**：极速！职业级响应
  - 🎯 **< 40ms**：非常优秀
  - ✅ **< 50ms**：良好
  - ⚠️ **≥ 60ms**：普通/较慢
- **数据统计**：自动计算平均响应时间、最快记录、测试次数

### ⏱️ 基础按压测试
- **实时按压耗时显示**：动态展示当前按键的按压时间
- **高精度计时**：使用 `performance.now()` 实现微秒级精度
- **击键日志**：详细记录每次按键的性能分析数据


### 使用步骤：
1. 选择"急停专项测试"或"基础测试"模式
2. 按住 W/A/S/D 任意键，然后快速松开
3. 查看实时响应时间和性能评级
4. 点击"🔄 重置测试数据"按钮清空记录

## 📊 测试原理

测量的 `duration` 包含以下延迟：
```
duration = releaseTime - pressStartTime
```

- **releaseTime**：keyup 事件触发时的精确时间戳
- **pressStartTime**：第一个 keydown 事件的精确时间戳（已过滤 repeat）

这个时间包含了键盘硬件延迟、USB 轮询、系统处理和浏览器事件处理的全部延迟。
